# Contributing to HAT

First off, thanks for taking the time to contribute!

## How to Contribute

1. **Find an issue** - Check [open issues](https://github.com/jodli/ha-tools/issues) or create a new one
2. **Fork & clone** - Fork the repo and clone it locally
3. **Create a branch** - Make your changes in a new branch
4. **Submit a PR** - Open a pull request with a clear description

## Development Setup

```bash
# Install dependencies
uv sync --all-extras

# Install pre-commit hooks
uv run pre-commit install

# Run tests
uv run pytest
```

## Questions?

Open an issue - we're happy to help!
